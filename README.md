# UI-Spot-the-Difference

A small collection of “UI Spot the Difference” pages built with plain HTML. Each file is a standalone challenge you can open in your browser to compare UI variations and practice attention to detail.

## What’s in this repo
- Multiple HTML files named like: `UI Spot the Difference 1.html`, `... 12.html`
- Each file is intended to be opened directly (no build step required)
- MIT licensed

## How to run
- Download and run locally
